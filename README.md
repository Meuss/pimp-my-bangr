# Pimp my BANGR

Because your timesheet shouldn't look like it was built in 2003. Dark mode for ~~Aubep~~ BANGR.

## What you get

- 🌙 **Full Dark Mode** — Every element restyled — header, tables, inputs, buttons, totals. Your eyes will thank you.
- ⚡ **Toggle On/Off** — One click in the toolbar to switch between dark and original. Your preference is remembered.
- 🪶 **Zero Overhead** — Pure CSS injection — no JavaScript modifying the page, no tracking, no nonsense. Just vibes.

## Installation

Takes about 30 seconds. No Chrome Web Store needed.

1. **Download the extension** — Grab the latest release zip from this repo and unzip it somewhere you won't lose it.
2. **Open Chrome Extensions** — Navigate to `chrome://extensions` in your browser.
3. **Enable Developer Mode** — Flip the toggle in the top-right corner of the extensions page.
4. **Load Unpacked** — Click the "Load unpacked" button and select the `extension` folder you unzipped.
5. **Go fill your timesheet** — Navigate to ~~Aubep~~ BANGR and enjoy the darkness. 🌑

## Usage

Click the extension icon in your toolbar to toggle dark mode on or off.
